# iTem常用指令

- 新建标签：`command + t`
- 关闭标签：`command + w`
- 切换标签：`command + 数字` 或者 `command + 左右方向键`
- 切换全屏：`command + enter`

- 分屏
  - 垂直分屏：`command + d`
  - 水平分屏：`command + shift + d`
  - `command + option + 方向键` 或 `command + [` 或 `command + ]`
- 其他
  - 清除当前行：`ctrl + u`
  - 到行首：`ctrl + a`
  - 到行尾：`ctrl + e`
  - 删除到文本末尾：`ctrl + k`
  - 删除光标之前的单词：`ctrl + w`
  - 清屏：`command + r`
